AI 에이전트, 그대로

데스크톱의 개발 환경을 그대로 가져오니까 Claude, Codex, Gemini 같은 AI 코딩 에이전트도 모바일에서 바로 사용할 수 있습니다.

호스트 관리

여러 개발 서버를 등록하고 관리하세요. VS Code Tunnel과 serve-web 두 가지 연결 방식을 모두 지원합니다.

설치 없이 바로 연결

데스크톱에 별도 프로그램을 설치할 필요가 없습니다. VS Code의 기본 터널 기능만으로 연결되며, 인증은 모두 앱 안에서 처리됩니다.

생체 인증 앱 잠금

Face ID 또는 Touch ID로 앱을 보호하세요. 앱 시작 시, 백그라운드 복귀 시 등 잠금 타이밍을 세밀하게 설정할 수 있습니다.

Siri 단축어

"PocketCode에 연결"이라고 말하면 자주 사용하는 호스트에 바로 접속. 홈 화면에 단축어를 추가하여 원탭으로 개발 환경에 접근하세요.

연결 상태 모니터링

네트워크 상태를 실시간으로 감지하여 연결 끊김을 즉시 알려드립니다. 불안정한 연결은 자동으로 재연결을 시도합니다.
